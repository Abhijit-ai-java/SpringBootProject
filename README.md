Overview:
The Fitness Tracker App is a web-based application built using Spring Boot that helps users track their daily fitness activities such as workouts, calories, and goals.
It provides secure login, progress tracking, and personalized dashboards to help users stay fit and motivated.

Tech Stack:
Backend: Spring Boot (Java)
Database: MySQL
Frontend: JSP / HTML / CSS / Bootstrap(work in progress)
Build Tool: Maven

Features:
User registration and login
Add, update, and delete fitness activities
Track calories burned and consumed
View daily and weekly progress
REST APIs for all major operations

Database Design:
Main tables:
User: store User information.
Goal: stores user fitness goals.
Excercise: store users excercise details.
WorkoutLog: store workout logs.
workoutSession: stores workout exercise details.
